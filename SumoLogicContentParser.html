New Users

▼ Export the JSON content for your Sumo Logic item to a file

  1. Login to the Sumo Logic app
  2. Locate your desired item in either the Library or your Personal folder
    • The Library can be found as an item in menu on the left or in a tab at the top of the window
    • Your Personal folder can be found as an item in menu on the left or in the Library itself
    • The item can either be a saved search, dashboard, or folder (including its subfolders
  3. Click the details icon for that item and then click Export to open the Export Item dialog
  4. In the Export Item dialog, click Download to download the JSON content to a file
    • You can alternatively click Copy to copy the JSON content to the clipboard, but then you will have to paste that content into a file and save it

▼ Import your JSON content file into this app

  1. Click this button:
  2. Select your file and click Open
    • All information is locally processed in the browser

▼ Explore the configuration of your Sumo Logic item

  • Click on the Field Summaries button at the top of the window to see the hierarchical summaries of the JSON fields extracted from the item
    • An HTML or JSON Output Format can be selected
    • For the HTML Output only:
      • The items can be filtered by type by selecting or deselecting the Output Filter checkboxes
      • The fields displayed item can be shown or hidden using the Expand All Items and Collapse All Items buttons, respectively
  • Click on the Query Fields button at the top of the window to see the hierarchy of the item along with its query and extracted JSON fields
    • An HTML or JSON Output Format can be selected
    • For the HTML Output only:
      • The query displayed item can be shown or hidden using the Expand All Items and Collapse All Items buttons, respectively

▼ Check Log Message data against your Sumo Logic item

  1. In the Message section:
    1. Click the Add Message button
    2. Paste a copy of a JSON log message into the text box
      • The border of the text box will be red if the log message fails a JSON syntax check
    3. If desired, repeat the previous 2 steps to add additional sample log messages
    4. Select a log message by clicking its radio button
      • A log message's radio button will be disabled until it passes a JSON syntax check, which is indicated by a black text box border
  2. In the Items section:
    • The items can be filtered by type by selecting or deselecting the Filter checkboxes
  3. In the Comparison section:
    • The field comparison rows can be filtered by type by selecting or deselecting the Filter checkboxes
